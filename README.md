# Установка и запуск приложения
===============================

Инсталляция
-----------
- Клонируем репозиторий
- Переходим в папку с проектом
- Работаем с консолью:

    | npm install -g gulp bower npm
    | npm install
    | bower install
    
Запуск и работа
---------------
- Запуск приложения коммандой 
    
    | gulp serve

- Требуется разрешить приложению работать в Twitter
